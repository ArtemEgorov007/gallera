# 📷 Photo Library

Приложение для просмотра коллекций фотографий с фильтрацией по категориям и постраничной навигацией. Данные загружаются из REST API (MockAPI).

## 🛠 Стек

- **React 19** (Create React App)
- **Feature-Sliced Design** — архитектура разделена на слои: `app`, `pages`, `features`, `entities`, `shared`
- **MockAPI** — внешний REST API для хранения коллекций
- **CSS** (модульные стили без UI-библиотек)
- **PropTypes** — типизация пропсов компонентов

## ✨ Функциональность

- Отображение фотоколлекций: каждая коллекция — одно большое фото и три миниатюры
- **Фильтр по категориям**: Все / Море / Горы / Архитектура / Города
- **Поиск по названию** коллекции (фильтрация на клиенте)
- **Пагинация** — 5 страниц, по 3 коллекции на странице
- Загрузка данных с REST API с обработкой ошибок

## 🚀 Запуск

```bash
npm install
npm start
```

Откроется на [http://localhost:3000](http://localhost:3000)

## 📁 Структура

```
src/
├── app/               # Точка входа, глобальные стили
├── pages/main/        # Главная страница
├── features/
│   └── filter-collections/   # Фильтр категорий + поиск
├── entities/
│   └── collection/    # Компонент коллекции фото
└── shared/
    ├── api/           # Fetch-функция для MockAPI
    ├── ui/Pagination/ # Компонент пагинации
    └── lib/helpers.js
```
